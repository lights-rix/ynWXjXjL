# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行PCB缺陷检测
- 内含6类PCB缺陷数据集：missing_hole、mouse_bite、open_circuit、short、spur、spurious_copper
- 提供500张PCB缺陷标注数据集
- 支持选择不同模型进行检测
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存功能
- 可自定义置信度和IOU阈值
- 可选择是否显示标签和原图

## 适用场景
- PCB制造过程中的质量控制
- 电子产品的质量检测
- 电路板缺陷的自动化识别
- 工业生产线上的实时检测

## 技术特点
- 基于YOLOv8目标检测算法
- 采用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 使用Python3.9、opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供完整的UI界面源文件和图标
- 包含完整的项目运行教程
- 环境配置成功后可直接使用
